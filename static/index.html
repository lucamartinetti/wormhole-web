How it works

wormhole.page is a Magic Wormhole client that runs entirely in your browser via WebAssembly. Encryption keys are negotiated client-side using SPAKE2 and all data is encrypted with NaCl SecretBox. The server never sees your files.

Fully interoperable with all Magic Wormhole clients — send from your browser, receive on the command line, or vice versa.

Compatible apps

wormhole-rs (Rust CLI) magic-wormhole (Python CLI) wormhole-william (Go CLI) Wormhole File Transfer (iOS) Wormhole William (Android) iyox Wormhole (Android)
